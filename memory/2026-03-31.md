**commit:** 8b4f744

**features implemented:**
1. **Auto-Logging** - Every trade entry/exit automatically logged to JSON
2. **P&L Tracking** - Dollar and percentage P&L calculated on exit
3. **Duration Tracking** - Time held calculated in minutes
4. **Analytics Dashboard** - Win rate, avg P&L, total P&L, max drawdown streak
5. **Best/Worst Trades** - Highlights top and bottom performers
6. **Period Selection** - Today/Week/Month/All time filters
7. **Trade Tags** - Support for strategy tags and manual notes
8. **CSV Export** - One-click export for tax/accounting
9. **Journal Panel** - Complete trade history viewable in dashboard

**journal entry format:**
```json
{
  "trade_id": "trade_20260331_120000_SPY",
  "status": "closed",
  "entry": {
    "timestamp": "2026-03-31T12:00:00Z",
    "symbol": "SPY",
    "option_type": "PUT",
    "strike": 400,
    "expiration": "2026-03-31",
    "price": 0.01,
    "quantity": 1
  },
  "exit": {
    "timestamp": "2026-03-31T12:30:00Z",
    "price": 0.02,
    "reason": "manual"
  },
  "pnl": {
    "dollar": 1.00,
    "percent": 100.0
  },
  "duration_minutes": 30,
  "signal_source": "scanner",
  "tags": ["0DTE", "momentum"]
}
```

**analytics shown:**
- Total trades count
- Win rate percentage (color-coded)
- Total P&L (color-coded)
- Average P&L per trade
- Max loss streak (drawdown)
- Average trade duration
- Best trade (symbol + P&L)
- Worst trade (symbol + P&L)
