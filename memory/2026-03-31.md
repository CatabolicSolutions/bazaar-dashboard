
## 2026-03-31 - Portfolio Heatmap

### DONE
**commit:** 0f65c6e

**features implemented:**
1. **Visual Heatmap Grid** - Each position as a colored cell
2. **Size = Position Value** - Larger cells = bigger positions
3. **Color = P&L** - Green winners, red losers, intensity = magnitude
4. **Risk Metrics Panel** - Total delta, theta, vega, max loss scenario
5. **Concentration Bars** - Visual % allocation by symbol
6. **Risk Alerts** - Automatic warnings for concentration/directional bias

**heatmap cells show:**
- Symbol (large text)
- Strike price + DTE
- P&L percentage (color-coded)
- Cell size relative to portfolio value

**risk metrics displayed:**
- **Total Delta** - Portfolio directional bias (Bullish📈/Bearish📉/Neutral)
- **Daily Theta** - Time decay per day ($/day)
- **Total Vega** - IV sensitivity
- **Max Loss** - Worst case if all positions go to zero

**risk alerts:**
- ⚠️ Concentration: Symbol >20% of portfolio
- ⚠️ Directional Bias: Net delta >±50 (too directional)
- ℹ️ Theta Burn: Losing >$100/day to time decay

**concentration visualization:**
- Horizontal bars showing % per symbol
- Red highlight if >20%
- Real-time updates

**demo:**
- 3 positions: SPY PUT (green +15%), TSLA CALL (red -8%), AAPL PUT (neutral 0%)
- Risk metrics: Delta -25 (Bearish), Theta -$45/day
- Alert: "TSLA is 35% of portfolio (max 20%)"
- Alert: "Portfolio is highly bearish (delta: -25)"
