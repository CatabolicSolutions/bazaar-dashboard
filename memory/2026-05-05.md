# 2026-05-05

## HQ Dashboard Evolution (Bloc HQ) — Overnight

Major overhaul of the dual rotator HQ dashboard on Bazaar VPS (137.184.144.196).

### Backend (dual_api_server.py)
- Added SSE streaming endpoint at `/api/hq/stream`
- Every endpoint returns `seq` field for sequence tracking
- SSE pushes: state (with decision matrix), health, heartbeat
- `compute_decision_matrix()` now fetches live prices via `get_live_prices()` from dual_rotator
- Decision matrix returns per-asset (ETH/BTC): vol_gate, convergence, trend, discount, reentry_score (weighted: D×0.5 + C×0.3 + T×0.2), thresholds, and raw values for each gate
- Added cycle_ledger.jsonl / bloc_trace.jsonl / arb_log.jsonl / scorecard parsing
- Health endpoint parses vigil report

### Frontend (hq_dashboard_v2.html)
- **SSE with fallback polling** — EventSource connects to `/api/hq/stream`, falls back to 3s polling on disconnect
- **Sequence tracking** — status bar shows last seq ID
- **Tabbed layout** — Overview | Analytics | Arb | Ops
- **Clickable gate cards** — each of the 4 gates (Vol Gate, Conv, Trend, Discount) shows ✓/✗, value, hint; click opens modal with raw values
- **Decision summary** — best re-entry score displayed in panel header with gauze + weight labels
- **P&L chart** — cumulative P&L from trades (reversed order), positive/negative fill
- **Drawdown chart** — peak-to-trough drawdown curve with max DD label
- **Arb chart** — spread_bps line chart with zero line
- **Bot state card** — side badge, signal, entry/peak, P&L, trade count, hold/flip bars
- **Execution feed** — color-coded (BUY=green, STOP=red, ROLL=yellow)
- **Drill modal** — tabbed (Overview + Raw), shows P&L, entry/exit class, path, tx hash, snapshot of current state
- **Responsive** — single-column on <1024px
